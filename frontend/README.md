# GPTlike Chat Frontend

Microsoft公式サンプル（sample-app-aoai-chatGPT）のスタイルを参考にしたチャットUIです。

## 開発

### 開発サーバーを起動

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開きます。

### ビルド

```bash
npm run build
```

ビルドされたファイルは `../static/` に出力されます。

## 機能

- 💬 リアルタイムチャット
- 📄 ドキュメントロード機能
- 🤖 エージェンティックRAG統合
- 📱 レスポンシブデザイン
- 🎨 Fluent UI コンポーネント
- ✨ Markdown対応

## 技術スタック

- React 18
- TypeScript
- Vite
- Fluent UI
- React Markdown
